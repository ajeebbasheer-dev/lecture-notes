================
Getting Started
================

.. list-table:: Frozen Delights!
   :widths: 15 10 30
   :header-rows: 1

   * - Name
     - Type
     - Description
     - Example
   * - Integers
     - int
     - whole numbers
     - 1, 2, 50, 100, 100000, etc.
   * - Floating point
     - float
     - numbers with decimal points
     - 1.2, 5.6, 9.3, 234.54, etc.
   * - String
     - str
     - ordered sequence of characters
     - "Hello", "", "foo", "54wfdghrdf-", etc.
   * - Lists
     - list
     - ordered sequence of **objects**
     - ["Hello", 234.1, 1002], [], [1, 2, 3], ['a', 'b', 's'], etc.
   * - Dictionaries
     - dict
     - un-ordered key-value pairs
     - {'name': 'john', 'age': 10}, {}, {1:2, 2:3}, etc.
   * - Tuples
     - tuple
     - ordered **immmutable** sequence of **objects**
     - ("Hello", 112, 12.3)
